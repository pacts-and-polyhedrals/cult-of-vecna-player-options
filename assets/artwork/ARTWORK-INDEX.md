# Vecna-Warped Arcana Artwork Index

This folder contains in-game artwork for the Vecna-Warped Arcana module.

## Folder Layout

- `book-art/`: square creature book art used by Actor sheets.
- `token-art/`: circular creature token art used by Actor prototype tokens.
- `spell-icons/`: one spell icon for every spell item in the module.
- `sprite-sheets/`: contact sheets for quickly reviewing the token and spell icon sets.

## Spell Art

There are 44 spell icon files, one for each spell in the `Vecna-Warped Spells` compendium.

The compendium records are wired to these files using module-relative Foundry paths such as:

`modules/vecna-warped-arcana/assets/artwork/spell-icons/root-sight.png`

Actor records are wired to creature art using paths such as:

`modules/vecna-warped-arcana/assets/artwork/book-art/diane-clearwater-orthogenist-lich.png`

and prototype token paths such as:

`modules/vecna-warped-arcana/assets/artwork/token-art/diane-clearwater-orthogenist-lich-token.png`
