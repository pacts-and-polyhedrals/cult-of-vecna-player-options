# The Ninth School

A Foundry VTT module for D&D 5e. Created by **DM Asmo** / [Pacts and Polyhedrals](https://github.com/pacts-and-polyhedrals).

> *Nature is not sacred. Nature is unfinished. Druids preserve the error. Wizards correct it.*

## Install via Manifest URL

```
https://raw.githubusercontent.com/pacts-and-polyhedrals/the-ninth-school/main/module.json
```

In Foundry: **Settings → Add-on Modules → Install Module** → paste the URL above.

### The Eight Doors of Knowing

The full poem collection by Diane Clearwater — eight poems (one per school of magic), preface, and closing fragment. The hidden closing line of each poem is bolded for DM reference. Reading them in sequence is the surface layer of the doctrine. The true hidden eight-part doctrine of Orthogenesis requires an Arcana DC 17 check.

When any player rolls a natural 17+ on an Arcana check, the module automatically whispers the decoded doctrine to the GM in chat.
